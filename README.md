# feedy

> Scrape RSS feeds and add articles to pocket

## Getting started

Clone repo and download dependencies:

```bash
git clone git@github.com:eirsyl/feedy.git ${GOPATH}/src/github.com/eirsyl/feedy
cd ${GOPATH}/src/github.com/eirsyl/feedy
make vendor
```

Build application:

```bash
make build
```
